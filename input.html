Basic Input Types

Type Description
text Single-line text input (default if no type specified).
password Text field that hides input characters.
email Field for entering email addresses (with validation).
number Field for numeric input (can set min, max, step).
tel Field for telephone numbers (no validation, but mobile keyboards show dial pad).
url Field for entering a web address (validated format).
search Text field styled for search boxes (may include "clear" button).

Date and Time Inputs

Type Description
date Select a date (year, month, day).
month Select a month and year (no day).
week Select a week and year.
time Select a time (hour and minute).
datetime-local Select date and time (local, no timezone).

Choice Inputs

Type Description
checkbox Allows selecting one or more options.
radio Allows selecting only one option from a group (same name).
range Slider for numeric range (min, max, step).

Form Control Inputs

Type Description
submit Submits the form.
reset Resets all form fields to initial values.
button Custom button (no default action; use with JS).
hidden Hidden field (not visible to user; used to store data).
