üß© Session 3: Styling & Animating (CSS)

1 hour

üß† Topics Covered:

  • Add a CSS file (style.css) and connect it to HTML
  • Apply:
    • Colors, fonts, and padding
    • Backgrounds (solid, gradient, or image)
    • Borders, shadows, and rounded corners
  • Use hover effects and animations to bring it to life

üíª Example CSS:

body {
  font-family: 'Comic Sans MS';
  background: linear-gradient(to bottom, skyblue, lightyellow);
  margin: 0;
}

header {
  background: royalblue;
  color: white;
  text-align: center;
  padding: 20px;
  position: sticky;
  top: 0;
}

nav a {
  color: white;
  text-decoration: none;
  margin: 0 10px;
  transition: color 0.3s, transform 0.3s;
}

nav a:hover {
  color: yellow;
  transform: scale(1.2);
}

img {
  border-radius: 10px;
  margin: 10px;
  transition: transform 0.5s;
}

img:hover {
  transform: scale(1.1);
}

footer {
  background: royalblue;
  color: white;
  text-align: center;
  padding: 10px;
  margin-top: 20px;
}

h2 {
  color: darkblue;
}

@keyframes glow {
  from { text-shadow: 0 0 5px white; }
  to { text-shadow: 0 0 20px yellow; }
}

h1 {
  animation: glow 2s infinite alternate;
}

üé® Add-On Challenges:

üß† Mini Quiz

Q1: What tag is used for navigation links?

Answer: The <nav> tag is used for navigation links.

Q2: What CSS property is used to make an element fixed on the screen?

Answer: position: fixed; makes an element stay in the same place on the screen.

Q3: True or False: You can use both transitions and animations on the same element.

Answer: True! You can use both transitions and animations together.
