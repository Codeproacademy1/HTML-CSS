🧩 Session 1: CSS Transitions – Making Changes Smooth

1 hour

🧠 Topics Covered:

  • What are transitions? (When something changes its style slowly, not suddenly.)
  • How to use transition with properties like color, size, or background.
  • Syntax: transition: property duration timing-function delay;
  • Using :hover to trigger animations.

💻 Activity: Magic Button

Make a button grow and change color when hovered.

<!DOCTYPE html>
<html>
<head>
  <style>
    button {
      background-color: lightblue;
      border: none;
      padding: 15px 30px;
      font-size: 18px;
      border-radius: 10px;
      transition: background-color 0.5s, transform 0.5s;
    }
    button:hover {
      background-color: royalblue;
      transform: scale(1.2);
      color: white;
    }
  </style>
</head>
<body>
  <button>Hover me!</button>
</body>
</html>

🎮 Challenge:

Make the button rotate a little when hovered (transform: rotate(10deg);).

🧩 Session 2: Transformations – Move, Rotate, Scale

1 hour

🧠 Topics Covered:

  • transform property and its cool tricks:
    • translate(x, y) → move element
    • rotate(deg) → spin it
    • scale() → grow or shrink
    • skew() → tilt
  • Combine transformations for creative results
  • Real-life example: "Like stickers moving and spinning on a paper."

💻 Activity: Dancing Shapes

Create squares that move, rotate, and change size when hovered.

<!DOCTYPE html>
<html>
<head>
  <style>
    .square {
      width: 100px;
      height: 100px;
      background: coral;
      margin: 30px;
      display: inline-block;
      transition: transform 1s;
    }
    .square:hover {
      transform: rotate(45deg) scale(1.3) translate(20px, -20px);
    }
  </style>
</head>
<body>
  <div class="square"></div>
  <div class="square"></div>
  <div class="square"></div>
</body>
</html>

🧩 Session 3: CSS Animations – Bringing Pages to Life

1 hour

🧠 Topics Covered:

  • Difference between transition and animation
    • Transition = happens when something changes (like hover)
    • Animation = runs automatically or repeats
  • Using @keyframes to define animation steps
  • Animation properties:
    • animation-name
    • animation-duration
    • animation-iteration-count
    • animation-delay
    • animation-direction

💻 Activity: Bouncing Ball

Create a ball that bounces up and down.

<!DOCTYPE html>
<html>
<head>
  <style>
    .ball {
      width: 80px;
      height: 80px;
      background: red;
      border-radius: 50%;
      position: relative;
      animation: bounce 2s infinite ease-in-out;
    }

    @keyframes bounce {
      0% { top: 0px; }
      50% { top: 150px; }
      100% { top: 0px; }
    }
  </style>
</head>
<body>
  <div class="ball"></div>
</body>
</html>

🎨 Project of the Week: Mini Animation Show

💡 Objective:

Create a fun webpage where several elements move differently (like a short "cartoon").

💻 Example Project:

The Dancing Scene — Sun rises, clouds move, ball bounces, and text slides in.

<!DOCTYPE html>
<html>
<head>
  <style>
    body {
      background: skyblue;
      overflow: hidden;
      height: 400px;
      position: relative;
    }

    .sun {
      width: 80px;
      height: 80px;
      background: yellow;
      border-radius: 50%;
      position: absolute;
      left: 50px;
      animation: sunrise 5s infinite alternate;
    }

    @keyframes sunrise {
      from { top: 300px; }
      to { top: 50px; }
    }

    .cloud {
      width: 100px;
      height: 60px;
      background: white;
      border-radius: 50px;
      position: absolute;
      top: 80px;
      animation: moveCloud 10s linear infinite;
    }

    @keyframes moveCloud {
      from { left: -150px; }
      to { left: 100%; }
    }

    .text {
      position: absolute;
      bottom: 30px;
      width: 100%;
      text-align: center;
      font-size: 30px;
      color: white;
      animation: slideText 3s ease-in-out infinite alternate;
    }

    @keyframes slideText {
      from { letter-spacing: 2px; }
      to { letter-spacing: 10px; }
    }
  </style>
</head>
<body>
  <div class="sun"></div>
  <div class="cloud"></div>
  <div class="text">Welcome to Animation World!</div>
</body>
</html>
Welcome to Animation World!

🧠 Mini Quiz

Q1: What's the difference between transition and animation?

Answer: Transitions happen when something changes (like hover), while animations run automatically or can repeat.

Q2: Which CSS property makes an object rotate?

Answer: The transform: rotate(deg) property makes objects rotate.

Q3: What keyword do we use to define animation steps?

Answer: We use @keyframes to define animation steps.
