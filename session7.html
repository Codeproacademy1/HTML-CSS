🎯 Learning Objectives

By the end of this week, students will:

Style Backgrounds

Understand how to use background colors, images, and gradients in CSS.

Create Borders

Learn how to add borders and round the corners of elements.

Combine Styles

Combine background and border styles creatively.

Build Dream Card

Build a "My Dream Card" mini-project with background and border design.

🗓️ Session Plan (2 Hours Total)

🕐 Hour 1: Backgrounds and Borders

10 mins

1. Quick Review

Recap of Week 6:

  • How to change text color.
  • How to use Google Fonts.
  • How to use CSS properties like font-size, color, and text-align.

Ask quick questions:

"How can we make our text red?"

"What does font-family do?"

30 mins

2. CSS Backgrounds

Explain the different ways to style backgrounds.

Property Description Example
background-color Sets the background color background-color: lightblue;
background-image Adds a background image background-image: url("stars.png");
background-repeat Repeats or stops repeating the image background-repeat: no-repeat;
background-size Controls image size background-size: cover;
background-position Moves image position background-position: center;

🧩 Practice Exercise:

Create a colorful background:

<body>
  <h1>Welcome to My Colorful Page!</h1>
</body>

body {
  background-color: lightyellow;
  background-image: url("https://www.w3schools.com/css/img_tree.png");
  background-repeat: no-repeat;
  background-position: center;
  background-size: cover;
}
Welcome to My Colorful Page!

Let the kids experiment by changing colors and background images.

20 mins

3. Borders

Explain the 3 main border properties:

  • border-width
  • border-style (solid, dashed, dotted)
  • border-color

Show examples:

h1 {
  border: 3px solid red;
  padding: 10px;
}
This text has a red border!

Add rounded corners:

h1 {
  border-radius: 15px;
}
This text has rounded corners!

🎨 Fun Mini Exercise:

Ask students to make a "frame" around an image or paragraph using colorful borders and rounded corners.

🕑 Hour 2: Gradients and Mini Project

20 mins

4. CSS Gradients

Introduce gradients as "color blends."

Show linear gradient:

body {
  background: linear-gradient(to right, lightblue, pink);
}
Linear Gradient
Radial Gradient

And radial gradient:

body {
  background: radial-gradient(circle, yellow, orange);
}

🧩 Activity:

Kids try to make their own "rainbow" background using 3 or more colors:

background: linear-gradient(to right, red, orange, yellow, green, blue, purple);
Rainbow Gradient
30 mins

5. 🎯 Mini Project: "My Dream Card"

Each student creates a profile card that represents their dream job, hobby, or favorite character — using everything they've learned.

Project Requirements:

  • Use a nice gradient or image as background.
  • Add a border with rounded corners.
  • Include their name, dream title, and a short description.

Example:

<div class="card">
  <h2>Kareem the Web Designer</h2>
  <p>I create colorful websites and cool designs!</p>
</div>

body {
  background: linear-gradient(to bottom, #c9e4f7, #f7c9e4);
  font-family: Arial;
}

.card {
  background-color: white;
  width: 300px;
  margin: 50px auto;
  text-align: center;
  padding: 20px;
  border: 3px solid #333;
  border-radius: 20px;
  box-shadow: 5px 5px 10px rgba(0,0,0,0.2);
}

Kareem the Web Designer

I create colorful websites and cool designs! 🎨

Encourage creativity — they can add emojis, favorite colors, or an image of their dream.

10 mins

6. 🏁 Wrap-Up & Homework

Review what they learned:

"What's a gradient?" "How do you round the corners?"

📚 Homework

Create a "My Favorite Animal Card" with:

  • A background image
  • Border and gradient
  • Animal name and fun fact

Example Requirements:

  • Use at least one gradient (linear or radial)
  • Add a border with rounded corners
  • Include the animal's name and an interesting fact
  • Make it visually appealing with colors and styling

📚 Materials Needed

  • Computers or laptops with code editor (VS Code or CodePen)
  • Internet connection (for background images and gradients)
  • Color & gradient examples printed or displayed
  • Simple project handout ("My Dream Card" template)
