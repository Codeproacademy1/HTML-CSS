Term Meaning

üß© Step-by-Step Lesson

üé® Part 1: What is CSS? (15 minutes)

Explain:

"HTML builds the structure ‚Äî like the walls of a house. CSS is the paint, decorations, and style!"

Show the structure of a CSS rule:

selector {
  property: value;
}

Example:

h1 {
  color: blue;
}

üí° Explain:

  • h1 ‚Üí tells the browser what to style (selector).
  • color ‚Üí what you want to change (property).
  • blue ‚Üí how you want to change it (value).

üé® Part 2: Internal CSS (20 minutes)

Show how to add a <style> section inside the <head>:

<!DOCTYPE html>
<html>
  <head>
    <title>My Personal Website</title>
    <style>
      body {
        background-color: lightyellow;
      }

      h1 {
        color: darkblue;
        font-family: Arial;
      }

      p {
        color: darkgreen;
        font-size: 18px;
      }
    </style>
  </head>
  <body>
    <h1>Welcome to My Website</h1>
    <p>Hi! I'm Kareem. I love coding and learning new things!</p>
  </body>
</html>

Hi! I'm Kareem. I love coding and learning new things!

üí¨ Discuss:

  • background-color changes the whole page color.
  • color changes text color.
  • font-family changes the font.
  • font-size changes how big the text is.

üß† Fun Exercise:

"Change the background to your favorite color and make the text readable."

Encourage creativity but remind them:

"Bright text on bright backgrounds is hard to read!"

üß± Part 4: Inline vs Internal vs External CSS (10 minutes)

Show the three ways to use CSS:

Inline CSS (inside a tag)

<h1 style="color:red;">Hello</h1>

‚û§ Use only for quick changes.

Internal CSS (inside <style>)

<style> p { color: blue; } </style>

‚û§ Best for small projects.

External CSS (in another file)

<link rel="stylesheet" href="style.css">

‚û§ Best for big projects (will be used later in Week 6).

üí™ Part 5: Mini Project ‚Äî "My Styled Website" (15 minutes)

Goal: Add CSS to their "Personal Website" from Week 4.

‚úÖ Requirements:

  • Change the background color of the page.
  • Make the main heading (h1) a fun color.
  • Style paragraphs with a readable font.
  • Give each section a border and some padding.
  • Change link colors in the <nav> bar.

Example CSS:

body {
  background-color: #fef9e7;
  font-family: "Comic Sans MS";
}

header {
  background-color: #aed6f1;
  padding: 10px;
}

nav a {
  color: darkblue;
  font-weight: bold;
  text-decoration: none;
  margin-right: 10px;
}

footer {
  background-color: #d5f5e3;
  padding: 5px;
  text-align: center;
}

Encourage kids to test colors using color names (red, green, lightblue) or hex codes (optional).
