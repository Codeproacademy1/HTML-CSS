📚 Book Store Project

🧠 Purpose

This project teaches students how to build a simple web page using HTML. They’ll learn how to add titles, paragraphs, author information, book details, and images from the internet.

🧩 Project Description

This web page is about a small Book Store. It introduces a book called The Story of Egypt and gives some details about the author.

  • 🏷️ A title at the top — “Book store”
  • ✍️ Paragraphs describing the author and book
  • 📚 Two images showing book covers

🧱 HTML Structure

1. <!DOCTYPE html> — Declares HTML5.
2. <html> — Wraps all content.
3. <head> — Page information + CSS link.
4. <body> — Visible page content.
5. <h1> — Title of the page.
6. <p> — Paragraphs for text info.
7. <img> — Adds book images using links.

💻 Example HTML Code


<!DOCTYPE html>
<html>
  <head>
    <link rel="stylesheet" href="styles.css" />
  </head>

  <body>
    <h1 class="title">Book store</h1>
    <p class="Author1">About author of written the story of Egypt</p>
    <p class="Author2">His name Kareem Ashraf, He is graduated from Faculty of Art</p>
    <p class="mybook">My book talks about Egyptian culture</p>
    <p class="cost">Price is 200$</p>

    <img src="https://images-eu.ssl-images-amazon.com/images/I/71LPCoF-MAL._AC_UL210_SR210,210_.jpg" width="250">
    <img src="https://images-na.ssl-images-amazon.com/images/S/compressed.photo.goodreads.com/books/1444690560i/26889785.jpg" width="200">
  </body>
</html>
