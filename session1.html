üí¨ Warm-up (10 minutes)

Ask students:

  • "What websites do you visit every day?"
  • "What do all websites have in common?"

Explain simply:

"Every website is made with code. The most basic code is HTML."

Show a fun before/after:

Show a blank Notepad file ‚Üí then show the same code in a browser.

Explain: "The browser reads HTML and makes it look nice!"

üß© Step-by-Step Lesson (20 minutes)

Step 1: Create the HTML file

Tell students:

Open your code editor (like Replit, CodePen, or VS Code).

Create a new file called index.html.

Step 2: Type the structure

Explain each tag while typing slowly:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Web Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is my very first web page.</p>
  </body>
</html>

Step 3: Open it in a browser

Have them save the file and double-click it to open in Chrome.

Celebrate: üéâ "You just made your first website!"

üé® Guided Practice (20 minutes)

Let them customize their text:

Change the title to their name: <title>Kareem's Web Page</title>

Change heading to something fun:

<h1>Welcome to My World!</h1>
<p>I love coding, games, and cats!</p>

Encourage creativity:

"Add more paragraphs! Write about what you like to do after school."

Teacher walks around (or checks screens) and helps fix tag mistakes.

üèóÔ∏è Mini Project (20 minutes) ‚Äî "My Favorite Things"

Each student builds a personal webpage about their favorite things.

Goal: Use <h1> and at least three <p> paragraphs.

Example:

<!DOCTYPE html>
<html>
  <head>
    <title>My Favorite Things</title>
  </head>
  <body>
    <h1>My Favorite Things</h1>
    <p>I love drawing and building LEGO models.</p>
    <p>My favorite food is pizza.</p>
    <p>I enjoy watching funny videos on YouTube.</p>
  </body>
</html>

Optional fun add-ons:

  • Try <h2> for a smaller sub-heading.
  • Add emojis! (Kids love this üòÑ)

üéâ Show & Tell (10 minutes)

Ask 2-3 volunteers to show their pages on screen.

Praise creativity: "Nice use of headings!" "Cool writing style!"

Quick recap quiz (hands up):

  • "Which tag makes a big title?"
  • "What do we write inside <body>?"
  • "What is a paragraph tag?"

üè° Homework

Task: Add two more paragraphs to your "My Favorite Things" page and change the title to your name.

Optional bonus: Try using <h2> for a smaller heading.

<p>I play football every weekend.</p>
