🧩 Session 1: Understanding Positioning

1 hour

🧠 Topics Covered:

  • What does "position" mean in web design?
  • Default positioning: position: static;
  • Moving elements using:
    • position: relative;
    • position: absolute;
    • top, left, right, and bottom properties
  • Real-life analogy: "Putting stickers in different places on a board."

💻 Activity: Move the Box

Students will create 3 boxes and move them around using CSS positions.

<!DOCTYPE html>
<html>
<head>
  <style>
    .box1 {
      width: 100px;
      height: 100px;
      background-color: red;
      position: relative;
      left: 50px;
    }
    .box2 {
      width: 100px;
      height: 100px;
      background-color: blue;
      position: absolute;
      top: 50px;
      left: 150px;
    }
  </style>
</head>
<body>
  <div class="box1"></div>
  <div class="box2"></div>
</body>
</html>

🧩 Session 2: Exploring Fixed & Sticky Positions

1 hour

🧠 Topics Covered:

  • position: fixed; — stays in place when scrolling
  • position: sticky; — sticks when scrolling past it
  • When to use each
  • Example: Sticky header or navigation bar

💻 Activity: Sticky Header Page

Students create a simple web page with a sticky header.

<!DOCTYPE html>
<html>
<head>
  <style>
    body {
      height: 1200px;
      margin: 0;
    }
    header {
      position: sticky;
      top: 0;
      background: orange;
      color: white;
      padding: 15px;
      text-align: center;
    }
  </style>
</head>
<body>
  <header>🌟 I Stick to the Top!</header>
  <p>Scroll down to see me stay here!</p>
</body>
</html>

🧩 Session 3: Z-Index and Layering Fun

1 hour

🧠 Topics Covered:

  • What is z-index?
  • How layers work — front and back order
  • Using z-index with positioned elements

💻 Activity: Layered Art

Students create 3 overlapping circles and adjust their z-index to bring one in front or back.

<!DOCTYPE html>
<html>
<head>
  <style>
    .circle {
      width: 100px;
      height: 100px;
      border-radius: 50%;
      position: absolute;
    }
    .red { background: red; top: 50px; left: 50px; z-index: 1; }
    .green { background: green; top: 80px; left: 80px; z-index: 2; }
    .blue { background: blue; top: 110px; left: 110px; z-index: 3; }
  </style>
</head>
<body>
  <div class="circle red"></div>
  <div class="circle green"></div>
  <div class="circle blue"></div>
</body>
</html>

🎨 Project of the Week: Moving Clouds & Sun Animation

🧠 Objective:

Combine positioning, layering, and basic animation.

💻 Example Project:

Create a page with a sky, sun, and clouds that move across the screen.

<!DOCTYPE html>
<html>
<head>
  <style>
    body {
      background: skyblue;
      height: 300px;
      overflow: hidden;
    }
    .sun {
      position: absolute;
      top: 50px;
      left: 100px;
      width: 80px;
      height: 80px;
      background: yellow;
      border-radius: 50%;
      z-index: 3;
    }
    .cloud {
      position: absolute;
      top: 100px;
      width: 100px;
      height: 50px;
      background: white;
      border-radius: 50px;
      z-index: 2;
      animation: move 10s linear infinite;
    }
    @keyframes move {
      from { left: -120px; }
      to { left: 100%; }
    }
  </style>
</head>
<body>
  <div class="sun"></div>
  <div class="cloud" style="animation-delay: 0s;"></div>
  <div class="cloud" style="top: 130px; animation-delay: 5s;"></div>
</body>
</html>

✨ Add Fun:

  • Change cloud sizes
  • Add a bird image that flies across
  • Make the sun move up and down like sunrise and sunset

🧠 Mini Quiz

Q1: What does position: fixed; do?

Answer: It makes an element stay in the same position even when the page is scrolled.

Q2: Which property controls which element appears on top?

Answer: The z-index property controls the stacking order of elements.

Q3: True or False: position: static; allows you to move elements.

Answer: False. Static positioning is the default and doesn't allow movement with top, left, right, or bottom properties.
