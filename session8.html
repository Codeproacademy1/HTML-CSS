5Ô∏è‚É£ Box Sizing

Explain that when you add padding and borders, the box gets bigger.

To fix that, we can use:

Tell them: "This makes your box behave better ‚Äî all parts stay inside the size you set!"

Demo:

  width: 200px;
  height: 100px;
  border: 5px solid black;
  padding: 20px;
  box-sizing: border-box;
}

With box-sizing
