🎯 Learning Objectives

By the end of this week, students will:

Understand Display Types

Understand the difference between block, inline, and inline-block elements.

Use Display Property

Learn how to use the display property to control element behavior.

Basic Layout Techniques

Discover basic page layout techniques.

Introduction to Flexbox

Get an introduction to Flexbox for aligning elements.

Build Hobbies Gallery

Build a fun "My Hobbies Gallery" web page using display and Flexbox.

⏰ Total Duration: 2 Hours

🕐 Hour 1: Display Types & Layout Basics

10 mins

1️⃣ Warm-Up & Review

Start with a few fun review questions:

  • "Who remembers what padding does?"
  • "What's the difference between margin and border?"
  • "How do we make round corners?"

Then show today's topic:

"Now that we can style boxes, let's learn how to arrange them on the page — side by side, in rows, or in columns!"

20 mins

2️⃣ Understanding Block vs Inline Elements

Explain visually:

  • Block elements take the full width (e.g., <div>, <p>, <h1>).
  • Inline elements only take as much space as they need (e.g., <span>, <a>, <img>).

🧩 Example:

<h2>Block Elements Example:</h2>
<div style="background-color: lightblue;">Box 1</div>
<div style="background-color: pink;">Box 2</div>

<h2>Inline Elements Example:</h2>
<span style="background-color: lightgreen;">Apple</span>
<span style="background-color: yellow;">Banana</span>
<span style="background-color: orange;">Orange</span>

Block Elements Example:

Box 1
Box 2

Inline Elements Example:

Apple Banana Orange

🎨 Mini Activity:

Ask kids to experiment —

"What happens if you change a div to span?"

"What happens if you add display: inline-block;?"

20 mins

3️⃣ The display Property

Explain that CSS can change how elements behave.

Value Description Example
block Makes element behave like a block display: block;
inline Makes element behave like text display: inline;
inline-block Inline but allows width/height display: inline-block;
none Hides the element display: none;

🧩 Example:

<div class="box">Box 1</div>
<div class="box">Box 2</div>
<div class="box">Box 3</div>

.box {
  display: inline-block;
  background-color: lightblue;
  width: 100px;
  height: 100px;
  margin: 5px;
}
Box 1
Box 2
Box 3

Let students experiment by changing display values and observing results.

10 mins

4️⃣ Aligning Elements Using Text-Align

Before introducing Flexbox, show simple alignment tricks:

.container {
  text-align: center;
}

and

.box {
  display: inline-block;
}

🧠 Explain: "When we center text or boxes, it makes our design look neat!"

🕑 Hour 2: Intro to Flexbox + Mini Project

25 mins

5️⃣ Introduction to Flexbox

Now say:

"Flexbox is like magic — it helps us move boxes easily and align them perfectly!"

Explain the 3 main ideas:

  • Flexbox works on a container.
  • The children inside move and align automatically.
  • You can control direction, spacing, and alignment.

🧩 Example:

<div class="container">
  <div class="item">🍎</div>
  <div class="item">🍊</div>
  <div class="item">🍇</div>
</div>

.container {
  display: flex;
  justify-content: center;
  gap: 20px;
}

.item {
  background-color: lightgreen;
  padding: 20px;
  font-size: 30px;
  border-radius: 10px;
}
🍎
🍊
🍇

Explain:

  • display: flex; → activates Flexbox
  • justify-content: center; → centers items
  • gap: 20px; → adds space between boxes

🎨 Mini Challenge:

Ask students to align emojis horizontally and then vertically by adding flex-direction: column;.

🍎
🍊
🍇
30 mins

6️⃣ 🎯 Mini Project: "My Hobbies Gallery"

Each student creates a small gallery of their hobbies — with icons or images — displayed in a nice row using Flexbox.

Example HTML:

<h1>My Hobbies</h1>
<div class="gallery">
  <div class="hobby">⚽ Football</div>
  <div class="hobby">🎨 Drawing</div>
  <div class="hobby">🎮 Gaming</div>
</div>

Example CSS:

body {
  background: linear-gradient(to bottom, #e0f7fa, #ffe0f7);
  font-family: "Comic Sans MS";
  text-align: center;
}

.gallery {
  display: flex;
  justify-content: center;
  gap: 20px;
  margin-top: 30px;
}

.hobby {
  background-color: white;
  border: 2px solid #333;
  border-radius: 15px;
  padding: 15px 25px;
  box-shadow: 4px 4px 10px rgba(0,0,0,0.2);
  font-size: 20px;
}

My Hobbies

⚽ Football
🎨 Drawing
🎮 Gaming

✨ Optional Add-On:

Let kids add hover effects using:

.hobby:hover {
  background-color: lightblue;
  transform: scale(1.1);
}

Or allow them to use images instead of emojis.

10 mins

7️⃣ Wrap-Up & Homework

Review Questions:

  • What's the difference between block and inline elements?
  • What does display: flex; do?
  • How do we center elements with Flexbox?

📚 Homework

Homework Project:

👉 Create a "My Favorite Animals" Flex Gallery with at least 4 boxes using:

  • Flexbox
  • Background colors
  • Rounded corners
  • Hover effect

Example Requirements:

  • Use display: flex to arrange the animals
  • Apply different background colors for each animal card
  • Add rounded corners with border-radius
  • Include a hover effect that changes the card appearance
  • Use emojis or images to represent each animal

  • Pre-made emojis/images for hobbies
