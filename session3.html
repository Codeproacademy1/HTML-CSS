🎯 Learning Goals

By the end of this week, students will:

Add Images

Understand how to add images to their web pages using <img>.

Create Links

Create links using <a> to connect pages or visit other websites.

Organize Files

Learn how to organize project files in folders (HTML + images).

Build Profile Page

Build a "My Profile Page" that includes their photo, bio, and favorite website links.

🧠 Key Vocabulary

Term Meaning
<img> HTML tag to show an image
src The image's file path (source)
alt Text shown if the image doesn't load
<a> Anchor tag — creates a link
href The web address or file path to link to
File path The location of a file in your project (like images/me.jpg)
Folder A place to store files (helps keep your project neat)

💬 Warm-up (10 minutes)

Ask:

  • "Who remembers what <ul> does?"
  • "What does <strong> do?"
  • "What do you think a website uses to show pictures?"

Explain:

"Today, you'll make your page look alive with images and links — just like real websites!"

Show examples from familiar sites (like a news site with images and links).

🧩 Step-by-Step Lesson

🖼️ Part 1: Adding Images (15 minutes)

Explain:

"The <img> tag shows pictures on your webpage. It doesn't need a closing tag!"

Syntax:

<img src="images/myphoto.jpg" alt="My photo">

Break it down:

  • src → tells the browser where the image is.
  • alt → describes the image if it doesn't load.
  • No closing tag — it's self-contained!

Example:

<h1>My Pet</h1>
<img src="images/cat.jpg" alt="A cute cat">
<p>This is my cat, Luna. She loves sleeping all day!</p>

My Pet

This is my cat, Luna. She loves sleeping all day!

💡 Tip for Kids:

  • Use a picture of themselves, their pet, or something they love.
  • If they don't have one, let them search free images on Unsplash or Pexels (safe for classroom use).

🌍 Part 2: Adding Links (20 minutes)

Explain:

"Links help us move around the web — from one page to another."

Syntax:

<a href="https://www.google.com">Visit Google</a>

Show:

  • The text between <a> and </a> is clickable.
  • The href attribute is the address to go to.

Example:

<h2>My Favorite Websites</h2>
<ul>
  <li><a href="https://www.youtube.com">YouTube</a></li>
  <li><a href="https://www.wikipedia.org">Wikipedia</a></li>
  <li><a href="https://scratch.mit.edu">Scratch</a></li>
</ul>

My Favorite Websites

  • YouTube
  • Wikipedia
  • Scratch

Internal Links (for later projects):

"We can also link to another HTML page in our own project!"

<a href="about.html">About Me</a>

💡 Tip: Explain "external" vs "internal" links with a simple analogy:

"External is going outside your house; internal is moving to another room."

🗂️ Part 3: Organizing Folders (15 minutes)

Explain:

"Good web developers keep their projects organized. Let's create folders to store HTML files and images neatly."

Show folder example:

my_website/
index.html
about.html
images/
me.jpg
cat.jpg
logo.png

Demonstrate:

  • Create a new folder called images.
  • Move all image files inside it.
  • Update code to use relative paths like:
<img src="images/me.jpg" alt="My Photo">

Explain relative paths in kid-friendly terms:

"If your HTML file is the house, your images are in the next room. You have to tell the browser where to find them!"

🏗️ Mini Project (10–15 minutes) — "My Profile Page"

Goal: Create a simple profile webpage that includes:

  • One profile photo
  • A short bio paragraph
  • A list of favorite websites (with clickable links)

Example:

<!DOCTYPE html>
<html>
  <head>
    <title>My Profile Page</title>
  </head>
  <body>
    <h1>My Profile</h1>
    <img src="images/me.jpg" alt="My picture" width="200">

    <h2>About Me</h2>
    <p>Hi, I'm <strong>Kareem</strong>. I'm 12 years old and I love coding, football, and pizza!</p>

    <h2>My Favorite Websites</h2>
    <ul>
      <li><a href="https://www.youtube.com">YouTube</a></li>
      <li><a href="https://www.wikipedia.org">Wikipedia</a></li>
      <li><a href="https://scratch.mit.edu">Scratch</a></li>
    </ul>
  </body>
</html>

Optional:

  • Add more images below using <img>.
  • Add a <hr> (horizontal line) to separate sections.

🎉 Show & Tell (5 minutes)

Have 2-3 students show their pages on screen.

Ask:

  • "What image did you use?"
  • "Which websites did you link to?"
  • "Can you explain what src and href mean?"

Encourage classmates to clap or give a "thumbs-up emoji" 👍 for each project.

🏡 Homework

  • Add two more links — one internal (to another HTML file they have) and one external (to a website they like).
  • Add at least two more images (like a hobby picture or a favorite place).
  • Make sure all images are inside an images folder.

🎯 Bonus Challenge (for fast learners)

Encourage them to experiment with:

Adding width and height to images:

<img src="images/pet.jpg" alt="My dog" width="150">

Making an image clickable:

<a href="https://www.youtube.com">
  <img src="images/youtube.png" alt="YouTube logo" width="80">
</a>

Linking their "All About Me" page (from Week 2) to their "Profile Page".

⚠️ Common Mistakes

Mistake What Happens Fix
Wrong image path Image doesn't show Check folder name & file extension
Forgot quotes around href or src Code error Always use " " around links
No alt text Bad accessibility Always add a description
Missing http:// or https:// in links Link won't open Include full URL

🧰 Teacher Tips

  • Prepare a folder of safe images for students who don't have their own.
  • Emphasize file names — avoid spaces (use underscores or dashes).
  • Show them how to drag-and-drop files into a folder in Replit or VS Code.
  • Reinforce the concept of "HTML is for structure; CSS is for style" (coming soon in Week 5).
