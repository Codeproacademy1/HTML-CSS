2. New Topic ‚Äì Text Properties

Introduce text styling properties step by step:

Property Description Example
color Changes text color color: red;
font-size Changes size of text font-size: 20px;
font-family Sets the text font font-family: Arial;
font-weight Makes text bold font-weight: bold;
text-align Aligns text (left, center, right) text-align: center;
text-decoration Adds underline/line-through text-decoration: underline;

üëâ Practice Exercise:

Students apply all these properties to style their name:

.my-name {
  color: darkblue;
  font-size: 40px;
  font-family: "Comic Sans MS";
  text-align: center;
  text-decoration: underline;
}
Kareem the Coder

Let students experiment with different colors, fonts, and alignments.

4. Adding Google Fonts

Introduce Google Fonts:

  • Go to the site and pick a font.
  • Copy the <link> tag into HTML <head>.
  • Use it in CSS with font-family:.

Example:

<link href="https://fonts.googleapis.com/css2?family=Pacifico&display=swap" rel="stylesheet">

h1 {
  font-family: "Pacifico", cursive;
}

This text uses the Pacifico font from Google Fonts!

üìö Homework

Requirements:

  • Use at least 3 different text properties (color, font-family, font-size, etc.)

  • Make sure the text is readable against the background
